Host, save, and archive documents on your own server!

This website allows you to host your documents on a centeral server, interact with the documents, edit them, and archive them. You can also restrict access to the documents, discuss the documents, and use Markdown!

There is a API that you can interact with outside of the website to view and edit the documents! See more about this on the GitHub below.

Although you might not be able to use this instance of the website. You can host your own, more documentation comming soon!

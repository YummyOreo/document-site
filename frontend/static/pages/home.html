Host, save, and archive documents on your own server!

This website allows you to host your documents on a centeral server, interact with the documents, edit them, and archive them. You can also restrict access to the documents, discuss the documents, and use Markdown!

There is a API that you can interact with outside of the website to view and edit the documents! See more about this on the GitHub below.

Although you might not be able to use this instance of the website. You can host your own, more documentation comming soon!

More info on these featurs below!

This is a concept and is subject to change

Features

Hosting

Right now there is no documentation on hosting. But there will be soon, so stay tuned for hosting documentation. It will be released on the GitHub below.

Making Documents

You can see this feateure on the navbar right now. Just click the plus icon on the navbar and it will walk you through how to create a document!

Editing Documents

Currently this feature has not been idmplemented. This will allow people that have access to edit the document, to edit it!

Archiving

Archiving is a feature that allows you to save old documents, for a long time, without anyone editing it! This feature will cause the document to disable comments and editing.

Viewing/Restricting

Restricting allows you to restrict who can: make documents; view, edit, and comment on documents. For this project we will use disocrd to restrict access.
